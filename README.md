# Disfluency
Disfluency es una app creada para facilitar el tratamiento de disfluencias entre terapeutas y pacientes. Las personas con disfluencias en el habla suelen tratarse con fonoaudiólogos que no siempre disponen de las herramientas ideales para el tratamiento óptimo de esta terapia. El objetivo de Disfluency es proporcionar una herramienta complementaria en la terapia que facilite el trabajo del fonoaudiólogo y fomente la práctica del paciente.
